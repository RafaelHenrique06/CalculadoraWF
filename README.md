Calculadora Windows For App em C#

Esta é uma calculadora desenvolvida em C# utilizando Windows Forms.
O sistema realiza operações matemáticas básicas como soma, subtração, multiplicação e divisão.

Projeto criado para praticar lógica de programação e manipulação de eventos no C#.

Funcionalidades:

- ✔ Soma
- ✔ Subtração
- ✔ Multiplicação
- ✔ Divisão



Tecnologias:

- C#
- .NET
- Windows Forms

  Como executar o projeto:
  
1. Clone o repositório
2. Abra no Visual Studio
3. Compile e execute (F5)
